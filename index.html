Учебные мини-проекты

Эта страница

Визуализация списка мини-проектов.

При разработке было использовано:

  • Настройка плагинов для комфортной верстки в VSCode (Live Server, Live Sass Compiler, ESLint, Auto Complete Tag и др.)
  • Анимация для фона страницы
  • CSS написан с использованием SASS
  • SASS разбит на разные логические файлы
  • Использован @mixin для кнопок
  • Потроганы CSS-функции min(), max(), clamp()
